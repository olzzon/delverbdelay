# DeliVerb

This is a true versatile AU Plugin, with a combined Delay, Reverb
The style button allows to go from Classic Deley-Reverb to Modern atmospheric sounds

# Features
On the front panel you have access to :
- Delay Time
- Delay Repeat
- Delay Mix
- Reverb Size
- Reverb Style (Classic to Modern)
- Reverb Mix
On the Advanced page you can access to :
- The same controls as base UI
- A Low Cut and High Cut filter for the Delay
- A Low Cut and High Cut filter for the Reverb
- 3 buttons for Ducking
  - Amount Delay: Amount of ducking applied to the Delay when the input signal is present
  - Amount Reverb: Amount of ducking applied to the Reverb when the input signal is present
  - Behaviour:  From low to high balance:
                Low: Duck Delay while playing, and duck Reverb when stopping playing
                High: Duck Reverb while playing, and duck Delay when stopping playing 

The style know does:
- Classic: Standard Delay + Reverb sound
- Modern: More atmospheric sound, with more diffusion on the Delay and Reverb
- Hybrid: Mix between Classic and Modern
- Atmospheric: Very diffused sound - great for pads and ambient sounds
- Cloudbursting: Very wet sound, with a lot of diffusion and octave effects
